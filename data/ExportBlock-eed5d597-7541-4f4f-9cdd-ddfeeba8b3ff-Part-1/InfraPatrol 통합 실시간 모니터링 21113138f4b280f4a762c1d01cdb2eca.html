InfraPatrol: 통합 실시간 모니터링

프로젝트시스네트정보
기간2017년 8월 - 2019년 12월
기술웹디자인/웹퍼블리싱

실시간 센서 데이터를 처리 및 시각화하는 서버사이드 기반 웹 서비스 디자인 변경 및 기능 추가

1. 웹 디자인

  • 센서 데이터를 시각화하는 위젯 형태의 대시보드 UI 디자인 설계
  • 다양한 위젯 디자인
  • 장치·센서별 상태 및 통계 시각화 레이아웃 디자인 설계

2. 백엔드

  • CodeIgniter(PHP) 기반으로 센서 데이터 로드 및 처리 로직 구현
  • 실시간 데이터 흐름에 맞춰 센서 정보 및 이력 데이터 제공 API 구성

3. 프론트엔드

  • 실시간 센서 데이터를 다양한 위젯 형태로 시각화
  • 다양한 위젯 추가
  • 서버 렌더링 방식으로 데이터 갱신 및 상태 표현 최적화
