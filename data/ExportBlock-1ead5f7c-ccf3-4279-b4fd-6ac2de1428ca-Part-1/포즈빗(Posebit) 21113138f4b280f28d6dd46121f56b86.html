사용자의 자세와 착좌 시간을 실시간으로 감지해 앱으로 알람을 전달하는 센서 기반 스마트 의자 시스템

1. 회로 설계

  • ESP32를 중심으로, 자이로스코프와 ToF 센서를 포함한 회로 설계
  • LED, 사운드 출력 모듈을 연동하여 피드백 기능 구현
  • 충전 배터리 사용 설계
  • 배터리 충전 회로 및 전력 관리(전압 조절, 보호 회로 등) 설계 포함

2. 임베디드 소프트웨어

  • RTOS 기반으로 ESP32 펌웨어를 개발, 센서 제어 및 통신 로직 구현
  • BLE 통신 및 Wi-Fi 프로비저닝을 통해 모바일 앱과의 연동 기능 구현
  • AWS IoT Core와 MQTT 프로토콜을 활용한 클라우드 데이터 송수신 및 OTA 업데이트 기능 구현
  • AWS S3에서 파일 다운로드 기능 구현
  • ToF 센서 및 자이로스코프 데이터 처리 로직 구현
  • I2C, I2S, USART 등 다양한 하드웨어 인터페이스 사용
  • 저전력 통신을 위한 통신 프로토콜 개발
  • 저전력을 위한 전력 관리 로직 개발

3. 서버 및 데이터 처리 시스템

  • AWS IoT Core, AWS Amplify, AWS DynamoDB, AWS Analytics를 이용한 데이터 처리 및 저장

4. 모바일 앱

  • React Native 기반의 크로스 플랫폼 앱 개발
  • BLE 통신 및 Wi-Fi 프로비저닝을 통한 기기 연동 기능 구현
  • QR 코드 인식을 통한 기기 연결 프로세스 구현
  • AWS Amplify를 통한 백엔드 연동 및 인증 처리
  • Microsoft CodePush를 활용한 무중단 앱 업데이트 시스템 적용
  • iOS 및 Android 앱 스토어 배포

5. 하드웨어 금형

  • 의자 제조사와 함께 제품 설계 및 시제품 제작을 진행, 센서 내장 구조에 최적화된 형태를 공동 설계

6. 양산

  • 부품 선정 및 회로 검증 이후, 중국 제조 공장과 직접 협업하여 시제품 생산 및 양산 진행
  • 각 기기 별 QR 코드 스티커 제작
  • 리튬 폴리머 배터리 수입 진행
